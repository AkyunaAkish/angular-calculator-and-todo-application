# Simple Angular Calculator and Todo List Application

### To run on your machine:

```
fork/clone
$ npm install
$ bower install
$ npm install -g gulp
$ gulp
In the browser go to: localhost:3000
```

### Technologies used:

```
HTML5
AngularJS
Angular UI-Router
Skeleton.css
NodeJS/NPM
SCSS
GulpJS
Gulp-Sass
Gulp-Serve
```
